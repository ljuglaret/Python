On execute la suite d instructions suivante : x = x + y; y = x - y ; x = x - y Quels sont les contenus des variables x et y en fin de sequence? solution : x = x + y y = x - y Donc x = x + y - y Donc y = x x = x - y Donc x = x + y - x Donc x = y Cette suite d instructions permet d echanger les valeurs de deux variables. Inconvenient : Ce n est pas intuitif et inutilement long Solution : ```python tmp = x x = y y = tmp ``` En introduisant une troisieme variable "tmp" on peut stocker une copie de la variable "x" ainsi, on attribue a "x" la valeur contenue dans "y", et "y" peut recuperer le contenu de "x" par l intermediaire "tmp" ### La structure conditionnelle **Exemple : Indiquer si une annee est bissextile.** Pour rappel c est le cas si et seulement si : - l annee est divisible par 4 et non divisible par 100 - ou l annee est divisible par 400 ```python if ((x % 400 == 0) or ((x %100 != 0) and (x%4 == 0))) : print ("bla") else : print("pas bla") ``` ###Fonctions **Pourquoi** Pour eviter d ecrire plusieurs fois de suite le meme code **Exercices** - 1 : La fonction valeur absolue - 2 : Le maximum de trois nombres
